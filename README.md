Create virtual environment

```
python -m venv .venv
. ./.venv/bin/activate
```

Install Django

`python -m pip install Django`

Verify Installation

`python -m django --version`

---

Create new project

`django-admin startproject mysite`

Start development server

`python manage.py runserver`
